# 1.-Intelligent-Air-Quality-Prediction-and-Personalised-Health-Advisory-Using-Machine-Learning-Models
Developed a machine learning–based AQI prediction system using Linear Regression, Decision Tree, and Random Forest models on data from major Indian cities. Built an interactive Streamlit dashboard to visualize air quality trends and provide personalized health advisory recommendations.
This project focuses on developing an intelligent Air Quality Index (AQI) prediction system using machine learning techniques. The objective was to analyze historical air quality data from four major Indian cities and build predictive models capable of estimating AQI levels accurately.The workflow began with data preprocessing, including handling missing values, feature selection, and normalization. Feature engineering techniques were applied to improve model performance and capture pollutant relationships effectively. Multiple supervised learning models such as Linear Regression, Decision Tree, and Random Forest were trained and compared to identify the best-performing algorithm based on evaluation metrics.To make the system user-centric and practical, an interactive dashboard was developed using Streamlit. The dashboard visualizes AQI trends, pollutant levels, and predictions in an easy-to-understand format. Additionally, a personalized health advisory module was integrated, which provides precautionary recommendations based on predicted AQI levels and user health conditions (e.g., respiratory sensitivity).The project demonstrates the integration of machine learning, data visualization, and real-world environmental applications. It highlights how predictive analytics can support environmental monitoring and enable data-driven decision-making for public health awareness.
